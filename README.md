# XmasGame<br>
2020耶誕節小遊戲<br>
<a href="https://danielsung720.github.io/XmasGame">範例連結</a>

<br>
[需先更改]<br>
16行 button中的字串(ex. 聖誕節小遊戲 / 開始 / blabla....)<br>
78行 隊伍列表資訊 預設2隊<br>
94行 參加人員數量<br>
97行 題目種類<br>
<br>
[遊戲規則]<br>
點擊開始畫面中間的button後會進入遊戲<br>
由底下列表中最左邊的玩家開始答題<br>
題目是螢幕中間最上方的 (ex. 吉祥話)<br>
此時有5秒的作答時間<br>
裁判可透過右方的操作按鈕動作<br>
答對 => 輪到下一個玩家<br>
答錯 => 淘汰，並輪到下一個玩家<br>
下一題 => 轉換題目類型<br>
